cocos-lua中lua的class理解

  • 技术

1.obj是一个userdata
2.obj.a=1, a的值是存储在tolua.getpeer(obj)中的，可以用rawget(tolua.getpeer(obj), “a”)获取到
3.cc.Node是一个table，cc和cc.xx都是table
4.Node的方法都是存储在getmetatable(cc.Node)中的，可以用rawget(getmetatable(cc.Node), “setName”)获取到
5.getmetatable(obj) == getmetatable(cc.Node)
getmetatable(obj).index == getmetatable(cc.Node).index，且都是function
6.getmetatable(tolua.getpeer(obj)).__index == child, 且都是table，func1就是其中的值
